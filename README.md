## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/nagi-3d-portfolio.git
   cd nagi-3d-portfolio

2. Install dependencies:
    ```sh
    npm install

3. Start the development server:
    ```sh
    npm run dev
